## The building block

Ch 3 showed a single attention head. A full transformer block wraps multi-head
attention together with layer normalization, a feed-forward network, and
residual connections. Step through each stage to see what it does and why:

<TransformerBlock client:load />

<CacheCallout>

**The transformer block is the model's repeating unit.** Every block applies the same four operations — normalize, attend, normalize, transform — but each block has its own learned weights. The residual connections mean no block has to reconstruct the full representation; each one only needs to learn a small, targeted update. The next chapter shows what happens when you stack dozens of these blocks together.
