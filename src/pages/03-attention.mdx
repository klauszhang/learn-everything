## The problem attention solves

After embedding, every token has a vector — but each vector only knows about
itself. The word "it" has no idea what it refers to. "Bank" doesn't know if
it's near "river" or "savings." Attention is the mechanism that lets tokens
look at each other and gather context.

The intuition: each token asks a question ("what context do I need?"), then
scans every earlier token for relevant answers. The result is a weighted blend
— tokens whose answers best match the question contribute the most.

## Q, K, V — three roles per token

Each token's vector gets projected through three separate weight matrices,
producing three vectors with different jobs:

- **Query (Q)** — the token's question: *"What am I looking for?"*
  Think of it as a search query you type into a search engine.

- **Key (K)** — the token's label: *"Here's what I'm about."*
  Think of it as a document title in the search index.

- **Value (V)** — the token's payload: *"Here's what to take from me."*
  Think of it as the document content you actually read.

The dot product `Q · K` measures how well a query matches a key — like how
well your search query matches a document title. High dot product = strong
match = more attention. The model then takes a weighted average of the V
vectors, pulling more content from high-match tokens.

## Causal mask and attention weights

A token can only attend to tokens at its position or earlier — the **causal
mask** sets all future positions to −∞ before softmax. This is what makes
generation possible: each token has access to exactly the context it would
have during autoregressive decoding.

<CacheCallout>

**Attention is the transformer's context mechanism.** Before attention, each token's representation is independent — just its embedding. After attention, every token carries information from every other token it attended to. This is what gives the transformer its power: the same word gets a different representation depending on its context. Multi-head attention lets the model maintain several independent notions of "relevance" simultaneously, and the causal mask ensures generation remains autoregressive — each position can only look backward.
